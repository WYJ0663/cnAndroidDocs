提供允许你创建与输入法框架（IME）相似的拼写检查器的类。

要创建新的拼写检查器，你必须实现继承自 {@link android.service.textservice.SpellCheckerService} 的服务。 并继承 {@link android.service.textservice.SpellCheckerService.Session} 类， 基于界面提供的回调方法提供的文本提供拼写建议。在 {@link android.service.textservice.SpellCheckerService.Session} 回调方法中，你必须将拼写建议以 {@link android.view.textservice.SuggestionsInfo} 对象的形式返回。

包含拼写检查器的应用必须声明服务需要的 {@link android.Manifest.permission#BIND_TEXT_SERVICE}权限。 服务还要声明一个意图过滤器，以 {@code <action android:name="android.service.textservice.SpellCheckerService" />} 作为意图的动作，并应该包含 {@code <meta-data>} 元素，声明拼写检查器的配置信息。例如：

 <service android:label="@string/app_name" android:name=".SampleSpellCheckerService" android:permission="android.permission.BIND_TEXT_SERVICE" > <intent-filter > <action android:name="android.service.textservice.SpellCheckerService" /> </intent-filter> <meta-data android:name="android.view.textservice.scs" android:resource="@xml/spellchecker" /> </service>

示例代码参见 拼写检查器服务应用和示例代码 拼写检查器客户端应用。
