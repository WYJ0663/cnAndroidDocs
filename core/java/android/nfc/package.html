提供对近场通讯(NFC)功能的访问，允许应用读取 NFC 标签中的 NDEF 消息。 “标签”可以是显示为标签的另一个设备。

更多信息参见近场通讯开发者向导。

{@more}

这些类的概要如下：

{@link android.nfc.NfcManager}
高级别的管理器，用于获得该设备的 {@link android.nfc.NfcAdapter}。 你可以使用{@link android.content.Context#getSystemService}来获得其实例。
{@link android.nfc.NfcAdapter}
它代表设备的 NFC 适配器，是你执行 NFC 操作的入口点。你可以通过 {@link android.nfc.NfcManager#getDefaultAdapter} 或 {@link android.nfc.NfcAdapter#getDefaultAdapter(android.content.Context)}取得其实例。
{@link android.nfc.NdefMessage}
代表 NDEF 数据消息，是标准的“记录”格式，用于携带在设备和标签间传输的数据。 你的应用可以通过 {@link android.nfc.NfcAdapter#ACTION_TAG_DISCOVERED} 意图收到该消息。
{@link android.nfc.NdefRecord}
代表在 {@link android.nfc.NdefMessage} 中传输的一条记录，包含共享的数据以及对数据类型的描述。

注：不是所有的 Android 设备都提供 功能。
