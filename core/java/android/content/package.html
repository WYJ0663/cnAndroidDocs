包含在设备上访问和发布数据的类。主要包含三类API：

内容共享({@link android.content})
在应用组件间共享内容。最重要的类如下：
  • {@link android.content.ContentProvider} 和 {@link android.content.ContentResolver} 用于管理和发布与应用相关的持久数据。
  • {@link android.content.Intent} 和 {@link android.content.IntentFilter}， 用于在不同应用组件间传输结构化信息—允许组件启动其它组件并取得返回值。
包管理({@link android.content.pm})
用于访问Android包( {@code .apk})相关信息，包括其活动、权限、服务、签名和提供器的信息。 用于访问这些信息的最重要的类是{@link android.content.pm.PackageManager}。
资源管理({@link android.content.res})
用于取得关联到应用的资源数据，比如字符串、可绘制资源、多媒体和设备详细配置。 用于访问这些数据的最重要的类是{@link android.content.res.Resources}。
