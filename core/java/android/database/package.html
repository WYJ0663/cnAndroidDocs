包含用于为内容提供器返回数据的类。

如果你需要在私有数据库中管理数据，请使用{@link android.database.sqlite}类。 这些类用于管理从内容提供器查询返回的{@link android.database.Cursor}对象。 数据库一般由{@link android.content.Context#openOrCreateDatabase}创建和打开。 通过内容提供器创建请求，你可以使用{@link android.content.ContentResolver content.ContentResolver}类。

所有的数据库存储在设备的/data/data/<package_name>/databases目录中。
