包含用于应用管理私有数据库的 SQLite 数据库管理类。

应用使用这些类管理私有数据库。如果你创建内容提供器，你应该会使用这些类来创建和管理用于保存内容的数据库。 从内容提供器 开发者向导可以学到实现内容提供器的通常的方法。内容提供器的示例可以查看 SDK 示例应用NotePad中的NotePadProvider类。 Android提供的SQLite的版本为3.4.0。

如果你使用提供器发送给你的数据，则不需要使用这些SQLite类，应该使用通常的{@link android.database}类。

Android自带的sqlite3数据库工具在tools/目录中。你可以使用该工具查看或在设备上运行SQL命令。 在命令行窗口中输入sqlite3来运行该工具。
