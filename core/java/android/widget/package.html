widget 包包含用于应用程序屏幕上的用户界面(主要是可视的)元素。 你也可以设计自己的小部件。

要创建自己的小部件可以扩展 {@link android.view.View} 或其子类。 要在 XML 布局中使用你的小部件，你需要两个附加的文件。 这里列出了你实现自定义小部件时需要创建的文件：

  • Java 实现文件 - 给文件实现小部件的行为。 如果你从 XML 布局来实例化对象， 你也需要写构造函数的代码，用于从 XML 布局文件中接收所有的属性值。
  • XML 定义文件 - res/values/ 目录中的 XML 文件，定义用于实例化你的小部件的 XML 元素以及支持的属性。 其他应用程序会在他们的 XML 布局中使用这些元素和属性。
  • XML 布局 [可选]- res/layout/ 目录中的可选 XML 文件，描述你的小部件的布局。 你也可以在你的 Java 文件中做该工作。

ApiDemos 示例应用程序中，有创建自定义 XML 布局标签 LabelView 的示例。 查看如下文件，其展示了如何实现和使用自定义小部件：

  • LabelView.java - 实现文件
  • res/values/attrs.xml - 定义文件
  • res/layout/custom_view_1.xml - 布局文件
