该包中的类用于开发无障碍服务，提供代替或增强的用户反馈。

{@link android.accessibilityservice.AccessibilityService} 服务在后台运行，等待系统在发生 {@link android.view.accessibility.AccessibilityEvent} 事件时回调。一些事件在用户界面上显示一些状态变化， 比如焦点变更、按钮按下等等。服务可以请求“查询当前窗口中内容”的能力。 开发无障碍服务需要继承该类并实现其抽象方法。

{@link android.accessibilityservice.AccessibilityServiceInfo} 类描述了 {@link android.accessibilityservice.AccessibilityService} 服务。 在系统发生 {@link android.view.accessibility.AccessibilityEvent} 事件时， 根据该类中封装的信息来通知 {@link android.accessibilityservice.AccessibilityService} 服务。
