提供用于管理诸如查找设备、连接设备和管理与设备间的数据传输的蓝牙功能的类。

更多信息参见蓝牙开发者向导。

{@more}

蓝牙API使应用程序可以：

  • 搜索其它蓝牙设备
  • 查寻本地蓝牙适配器为蓝牙设备配对
  • 创建RFCOMM通道/接口
  • 连接其它设备的指定接口
  • 与其它设备互传数据

要使用这些 API 进行蓝牙通讯，应用必须声明{@link android.Manifest.permission#BLUETOOTH}权限。 一些附加功能，例如请求设备搜索，还需要{@link android.Manifest.permission#BLUETOOTH_ADMIN} 权限。

注： 不是所有的Android设备都提供蓝牙功能。
