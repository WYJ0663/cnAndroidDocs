这些类提供了属性动画系统的功能， 允许你生成任意类型的对象属性的动画。默认支持int、 float和十六进制颜色值。通过定制{@link android.animation.TypeEvaluator}、 告诉系统如何计算指定类型的值，你可以使用其它类型制作动画。

更多信息参见 动画开发者向导

{@more}

你可以设置不同类型的插值器 (包含在 {@link android.view.animation}中)、 指定{@link android.animation.Keyframe 关键帧}、按顺序或同时播放一组动画 (通过{@link android.animation.AnimatorSet}) 等操作，进一步控制动画播放。
